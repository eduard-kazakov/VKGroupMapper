VK GROUP MAPPER (Alpha v.0.001) - Помощь


VK Group Mapper - сервис, который поможет вам исследовать аудиторию вашей группы или публичной страницы ВКонтакте.

Географическое расселение участников различных сообществ - важный фактор при изучении особенностей пространства страны в целом. Лучший способ оценить географию расселения - составить тематическую карту. Именно эту задачу решает представленный сервис.

Для создания карты вы должны указать четыре параметра: ID группы, Тип картографирования, Способ разбиения всего диапазона значений на интервалы, Количество интервалов.

ID группы - это уникальный идентификатор группы или публичной страницы, который можно найти в адресной строке браузера при её посещении. Может быть числовым либо текстовым. Например, для группы "Институт наук о Земле СПбГУ" ID это слово geosciences.


Способов картографирования в текущей реализации сервиса предусмотрено два - Картограмма и Количественный фон. Картограмма предполагает вычисление для каждого из субъектов РФ относительного показателя - отношения количества участников группы, живущщих в этом субъекте, к общему числу его жителей. Показатель приводится к виду "количество участников на 1000 человек населения". Дальнейшее разбиение на диапазоны и цветовая дифференциация в таком случае зависят не от абстрактного числа людей, а от процентного охвата населения исследуемым сообществом. Количественный фон, в свою очередь, предполагает картографирование в зависимости от непосредственного количества участников из конкретного субъекта федерации, безотносительного общей численности населения, площади и иных факторов. Такой способ используется относительно редко, так как в большинстве случаев отражает нерепрезентативное состояние дел, ведь, условно, охват 1000 человек в Москве и на Чукотке - совсем по-разному характеризует ваше сообщество. Способов разбиения всего диапазона значений на интервалы предусмотрено два: равные интервалы и квантили. Разбиение на равные интервалы предполагает вычисление максимального и минимального значений картографируемого показателя среди всех субъектов РФ, а затем разделение интервала между ними на N равных по размеру частей (где N - количество интервалов). Например, существует набор значений [21,36,19,58,10,37,70,25, 39], который способом равных интервалов мы хотим разбить на 3 диапазона. Все значения лежат в пределах от 10 до 70, который разбивается на 3 равные части: 10 - 30; 30 - 50; 50 - 70. Теперь каждое отдельное значение из набора можно отнести к одной из этих частей.

Квантили, в свою очередь, это такой статистический способ разбиения на диапазоны, при котором в каждый из них попадёт одинаковое количество объектов. Для вышеприведенного набора значений разбиение окажется таким: 10 - 22 ; 22 - 38 ; 38 - 70. В каждую из этих частей попадет ровно по 3 элемента из набора.


После задания параметров и отправки их на обработку сервис начнёт собираеть данные с использованием API ВКонтакте и API Яндекс.карт. В случае, если численность группы или публичной страницы лежит в пределах до нескольких тысяч участников, построение карты займет непродолжительное время (5-20 секунд). Обработка больших же сообществ может занимать и несколько часов, в силу ограничений используемых API и больших объемов обрабатываемой информации.

Вам не обязательно ждать окончания процесса обработки - все запрошенные когда-либо карты после их создания доступны в разделе Архив карт.
